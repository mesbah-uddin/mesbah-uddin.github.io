3.3 Math formulae

Several ways of using math formulae in Tex.

There is some interesting discussion on this in this stackexchange submission.

Sigmoid activation function, \(\phi(z) = \frac{1}{1+e^{-z}}\) (simple inline representation)

For displaying the equation in a separate line use any of the following:

Sigmoid activation function, \[\phi(z) = \frac{1}{1+e^{-z}}\]

Sigmoid activation function,

\begin{displaymath} \phi(z) = \frac{1}{1+e^{-z}} \end{displaymath}

Sigmoid activation function,

\begin{equation} \phi(z) = \frac{1}{1+e^{-z}} \end{equation}

Sigmoid activation function,

\begin{gather*} \phi(z) = \frac{1}{1+e^{-z}} \end{gather*}

Sigmoid activation function,

\begin{align} \phi(z) = \frac{1}{1+e^{-z}} \end{align}
